## What is Fylepad?

Fylepad is a cross-platform notepad application designed for note-taking and Markdown-based editing. It strikes the perfect balance between simplicity and power, offering everything you need without the bloat.

**Lightweight and fast**: At just under 10MB, Fylepad is powered by a Rust core through Tauri, making it fast, secure, and memory-efficient. No bulky Chromium browser under the hood!

## Built with modern technology

Fylepad is built using cutting-edge technologies:

- **Nuxt.js**: Fast and modern web framework
- **Tauri**: Secure and lightweight desktop applications
- **TipTap**: Advanced rich text editing
- **Tailwind CSS**: Beautiful and responsive designs
