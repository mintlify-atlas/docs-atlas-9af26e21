Fylepad offers extensive styling options to customize the appearance of your notes. You can change fonts, apply colors, adjust text alignment, and format text to match your preferences.

## Accessing the styling panel

Open the styling panel by clicking the paint brush icon in the top toolbar. The panel appears as a bottom sheet on mobile or a popup on desktop.

<Note>
  Styling changes apply to selected text or the current paragraph. To style multiple paragraphs, select all the text you want to format before applying styles.
</Note>

## Font customization

Fylepad includes 39 carefully selected fonts ranging from serif to sans-serif, monospace, and decorative styles.

### Changing fonts

1. Open the styling panel
2. Click the font selector dropdown
3. Search for a font by typing in the search box
4. Click on a font name to apply it
5. The selected text will update immediately

### Resetting fonts

To return to the default font (Geist):

1. Open the styling panel
2. Click the trash icon next to the font search box
3. The text will revert to the default font

<Note>
  Font changes are saved with your document. When you reopen the note, your custom fonts will be preserved.
</Note>

## Color customization

Add visual emphasis to your text with custom colors.

### Applying text colors

1. Select the text you want to colorize
2. Open the styling panel
3. Click "Color" to expand the color picker
4. Click the colored circle to open the color selector
5. Choose a color from the picker or enter a hex code
6. The selected text will change color immediately

### Color picker features

The color picker supports:
- **Full RGB spectrum**: Choose from millions of colors
- **Hex input**: Enter specific color codes (e.g., #FF5733)
- **Visual preview**: See the current color before applying

### Resetting colors

To remove color formatting:

1. Select the colored text
2. Open the styling panel
3. Click the trash icon next to the color picker
4. The text will return to the default color

<Warning>
  When using colors, ensure sufficient contrast for readability. Low contrast can make text difficult to read, especially in different lighting conditions.
</Warning>

## Text alignment

Control how text is positioned within your document.

### Alignment options

Fylepad supports four text alignment modes:

1. **Left align** (default): Text aligns to the left margin
2. **Center align**: Text centers on the page
3. **Right align**: Text aligns to the right margin
4. **Justify**: Text spreads evenly between margins

### Changing alignment

1. Place your cursor in the paragraph you want to align
2. Open the styling panel
3. Click on the alignment buttons under "Text align"
4. The paragraph will realign immediately

<Note>
  Text alignment applies to entire paragraphs, not individual words or sentences.
</Note>

### When to use each alignment

<Accordion title="Left alignment">
  **Best for**: Body text, lists, most content
  
  Left alignment is the most readable for long-form text in Western languages. The consistent left edge provides a visual anchor for the reader's eyes.
</Accordion>

<Accordion title="Center alignment">
  **Best for**: Titles, headings, poems, quotes
  
  Center alignment draws attention and creates symmetry. Use sparingly for impact.
</Accordion>

<Accordion title="Right alignment">
  **Best for**: Dates, signatures, special formatting
  
  Right alignment is less common but useful for specific layout needs.
</Accordion>

<Accordion title="Justified alignment">
  **Best for**: Formal documents, publications, newspapers
  
  Justified text creates clean left and right edges. However, it can create awkward spacing in narrow columns.
</Accordion>

## Best practices for styling

<Note>
  **Be consistent**: Choose a small set of fonts and colors and use them consistently throughout your notes.
</Note>

<Note>
  **Don't overdo it**: Too many fonts and colors can make your notes look cluttered and unprofessional.
</Note>

<Note>
  **Consider accessibility**: Ensure adequate contrast between text and background colors. Avoid color as the only means of conveying information.
</Note>

<Note>
  **Match the purpose**: Use professional fonts for work documents, creative fonts for personal projects, and monospace fonts for technical content.
</Note>

## Styling persistence

All styling choices are saved automatically:

- **Fonts**: Preserved when saving, exporting, and reopening
- **Colors**: Maintained in both light and dark modes
- **Alignment**: Saved with paragraph formatting
- **Formatting**: Bold, italic, underline persist across sessions

### Export behavior

When exporting to Markdown:

- **Bold, italic, underline**: Converted to Markdown syntax
- **Colors**: Preserved as inline HTML when necessary
- **Fonts**: May not export (depending on target format)
- **Alignment**: May not export to standard Markdown

<Warning>
  Some styling options, particularly custom fonts and colors, may not be fully supported when importing your Markdown into other applications.
</Warning>

## Dark mode considerations

Fylepad automatically adjusts to your system's color scheme:

- Custom text colors remain visible in both modes
- Ensure your color choices work in both light and dark themes
- Test critical documents in both modes for readability
- Default text color automatically adapts to the theme

## Advanced tips

<Accordion title="Create style templates">
  For frequently used styles, create a template note with pre-formatted headings, body text, and highlights. Copy sections from this template when creating new notes.
</Accordion>

<Accordion title="Use semantic coloring">
  Assign meaning to specific colors (e.g., red for deadlines, green for completed items, blue for references) and use them consistently across all your notes.
</Accordion>

<Accordion title="Font pairing">
  Combine fonts thoughtfully. A common pattern is to use:
  - A sans-serif font (Inter, Helvetica) for headings
  - A serif font (Georgia, Playfair) for body text
  - A monospace font (Roboto Mono) for code
</Accordion>

<Accordion title="Experiment with emphasis">
  Instead of making text bigger or bolder, try using color or a different font to create emphasis. This can create more sophisticated visual hierarchy.
</Accordion>
